## Running Tests

```bash
# Unit tests (fast, no dependencies)
go test ./internal/... ./pkg/... -v -race -count=1

# Integration tests
go test ./test/integration/... -v -tags integration -count=1

# Scenario tests
go test ./test/scenarios/... -v -tags scenario -count=1

# Full pyramid (run all in order)
go test ./internal/... ./pkg/... -v -race -count=1 && \
  go test ./test/integration/... -v -tags integration -count=1 && \
  go test ./test/scenarios/... -v -tags scenario -count=1
```
